Safe Owner Subscriptions to the "My Safe Synthesis" Report

Do you need to give safe owners visibility into their safe access and configuration?

This table allows you to create and manage safe owner subscriptions to a report.

To activate a subscription:

  • click the "New Subscription" button,
  • select a safe owner from those offered, then
  • then select the safes that the person owns and should be notified about.

Upon each data upload, the person will receive an Excel report on all or part of the safes he owns by email.

The list of safes can be changed by clicking on the "Modify" button or the corresponding number of safes in the table.

The "Unsubscribe" button removes your subscription to the report.

At any time, reports can be manually emailed to a selected safe owner by using the "Send Report" button or to all subscribers by using the "Send All Reports" button.

If safe owners also have access to PAM Booster, they can change the scope of their subscription or stop it altogether by going to the last "Analytics" tab on the "My 360° Access" home page and selecting "My Reports Notification About My Safes."

"My Safes Synthesis" Excel Report

This report gives you a summary of your safes. It allows you to carry out the first level of necessary controls within the framework of the security policy.

It contains information about user access to your safes and their credentials. In addition, it shows the results of various verifications and security checks.

Additionally, a time-based analysis is conducted on the data to show any changes that have occurred since the last analysis was performed by PAM Booster.

The Excel report sent to safe owners includes seven tabs divided into three categories:

  • Summary

    • The tab named "Main" presents the summary of the Excel document.

  • Monitoring of My Safes

    • The tab named "SafeOwner KPIs" presents key indicators related to the selected safes.

  • My Safes Details

    • These five tabs present in a filterable table all the information about "safes," "users" having access to the owned safes, "groups" providing access to the selected safes, "credentials" secured in these safes, and finally, the list of control defects detected by PAM Booster.

Safe Owners:

When data is loaded, safe owners are automatically detected by PAM Booster.

By default, everyone with a "ManageSafe" authorization is considered to be the owner of the related safe.

It is possible to modify this rule during the installation of PAM Booster.

It is also possible to remove or add safe owners on the Safe Detail page.
